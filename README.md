# nextjs-real-time-app


simple real time chatting application

**server**

```
yarn install

yarn dev
```

**client**

```
yarn install

yarn dev
```

---


## Project Stack

- React 18
- Next.js
- TypeScript
- Styled-Components
- Recoil
- Socket.io
- Express
